# Playwright Automation Framework

Automation testing framework using Playwright (TypeScript) for UI and API testing.

## Features
- UI and API test support
- Modular structure (pages, utils, fixtures)
- HTML reporting
- Debugging with traces and videos

## Run tests

Run all tests:
npm test

Run UI tests:
npm run test:ui

Run API tests:
npm run test:api

Open HTML report:
npm run report
